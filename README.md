# wifi-stealer 🔑📡

## Description
The wifi-stealer project is a simple Batch script that allows users to obtain and save WiFi passwords from your computer or school computer🥷

## Features
- The script allows browsing and selecting available WiFi networks 🌐.
- Users can choose the network from which they want to retrieve the password 🔍.
- Upon user consent, the script displays the password or saves it to a file 💾.
- Provides a user-friendly interface for ease of use 🎨.
- Supports saving multiple passwords for future reference 📋.


## How to Use
1. Run the `wifi_stealer.bat` file.
2. Select the WiFi network from which you want to retrieve the password.
3. The Script will show the wifi password.
4. And Your Done!
